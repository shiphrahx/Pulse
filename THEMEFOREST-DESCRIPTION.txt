FULL DESCRIPTION
----------------

Pulse is a modern, professional portfolio and vCard template built with the latest web technologies. Showcase your work, skills, and expertise with this feature-rich, fully responsive single-page template.

Built with React 19, Vite 7, and Tailwind CSS 3.4, Pulse offers exceptional performance, clean code, and easy customization. Whether you're a developer, designer, freelancer, or creative professional, Pulse provides everything you need to create a stunning online presence.
